# EZ-USB&trade; FX20: USB Video Class (UVC) application 1.0.3

## What's Included?

Refer to the [README.md](./README.md).

## Feature Changes

* Updated application to make use of USBFXStack version 1.3.0
* Added an option to stream colorbar images based on data stored in the device RAM
* Added linker scripts and instructions to port the application to any EZ-USB&trade; FX device

## Defect Fixes

* Corrected clock and buffer size settings used to configure the PDM Microphone interface

## Supported Software and Tools
